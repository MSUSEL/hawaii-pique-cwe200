Help & Instructions

Step 1: Import Project

Click the Import button and select a zipped version of the project that you want to analyze.

Step 2: Upload Project

Click the Upload button to send your project to the backend for analysis.

Step 3: Analyze Project
  1. If this is the first time you are analyzing this project, or if you want to check if your changes have fixed previous vulnerabilities, click the Analyze button.
  2. If you have already run the analysis, click the History button to load previous results without additional cost.
    Unsure if the project has been analyzed? Check for the presence of a results.sarif file.
Step 4: View Results

If there are any vulnerabilities in the project, they will be displayed in the Rules tab on the right. Click on each rule to see exactly where the issue is located in the code.
