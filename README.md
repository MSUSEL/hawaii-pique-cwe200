### How do I get set up? ###

**In root directory**
1. Install dependencies
```bash 
npm run ci
```
2. Build backend and frontend
```bash
npm run build
```
3a. To launch backend:
```bash
npm run start-backend
```
3b. To launch frontend:
```bash
npm run start-frontend
```

**Note: ** 3a and 3b may be need to run in separate shells / consoles
